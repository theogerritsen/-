University of Lausanne

Climbing Web App

This is a mobile web application for finding climbing spots in the swiss Chablais. It uses Leaflet for the interactive map and OTP2 (Open Trip Planner) for the routing system.

University of Geneva

Landslide interactive map

This is an interactive map to show impacts of landslides throughout the years. It uses a Mapbox baselayer, OpenLayers for the interactivene map and D3.js for the interactive barplots.

University of Lausanne

Swiss Population interactive map

This is an interactive map that displays a swiss population indicator using mainly D3.js. Cantons are highlightes as you pass the cursos on them and barplots are shown if you click on a canton.

University of Lausanne

Augmented reality mobile web application

This is an augmented reality mobile web application built with Leaflet and AR.js. The user is able to do an augmented tour of Lausanne using only his mobile phone.
